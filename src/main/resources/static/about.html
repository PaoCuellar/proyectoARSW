¿QUIÉNES SOMOS?

Escuela Colombiana de Ingeniería Julio Garavito

Somos un grupo de estudiantes de la Escuela Colombiana de Ingeniería Julio Garavito cursando Arquitectura de Software, interesados en ofrecer una plataforma de subastas para personas dedicadas a la subasta y a fundaciones que necesitan de esta actividad para su subsistencia económica, ampliando la red de clientes.

OPORTUNIDADES

Personas que se dedican a las subastas

Este será un lugar seguro para tus ventas, acuerdas el medio de pago con el ganador de la subasta para más comodidad y menos cobros en la transacción.

Fundaciones

Las fundaciones tendrán un espacio seguro en el que ofrecerán los productos que poseen con la que recaudan sus fondos, mayor visibilidad, mayor cantidad de posibles clientes y acordar el medio de pago para no generar cobros de transacciones.

Compradores

Nuestros compradores, ganadores de las subastas, tendrán una comunicación directa con el subastador para acordar el pago ofreciendo seguridad en las transacciones y su legitimidad.

VENTAJAS

Subastas públicas

No queremos manejar ningún tipo de anonimato para tener interacción directa entre vendedor y comprador

Apoyo a fundaciones

Queremos que también sea una oportunidad para las fundaciones de tener una plataforma con mayor alcance que las redes sociales.

Sobre nosotros

Ofrecemos comunicación directa con el comprador para que sea mejor la experiencia de pago
